narrative: |
  ## Context
  Gap in the market: DeFi teams need reliable, verified onchain data but existing solutions are either too expensive or unverified.

  ## Approach
  Built a Python/FastAPI backend with PostgreSQL for data storage and Redis for caching. Implemented cryptographic verification of data feeds using Merkle proofs. Created a subscription billing system with Stripe.

  ## Outcome
  3 enterprise customers in closed beta. Generating MRR. Accepted to NinjaLabs research program for continued development.
